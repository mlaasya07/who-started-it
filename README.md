# who-started-it
Who Started It? is a web-based social network analysis system that models interpersonal relationships within small groups and applies machine learning and natural language processing techniques to infer social roles and interaction dynamics.


```
who-started-it/
│
├── index.html          # Landing / group creation
├── quiz.html           # MBTI + Zodiac entry
├── graph.html          # Chaos graph visualization
│
├── css/
│   └── style.css
│
├── js/
│   ├── data.js         # Shared state
│   ├── quiz.js         # Personality + zodiac logic
│   ├── graph.js        # Graph ML + role inference
│   └── nlp.js          # Sentiment / emotion scoring
```
